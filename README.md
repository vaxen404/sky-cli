# 🌤️ SKY-CLI HAVA DURUMU
Hava durumunu terminal üzerinden anlık ve renkli bir şekilde takip etmenizi sağlayan hızlı bir CLI aracı.

## 🚀 Özellikler
- Anlık Veri: OpenWeatherMap API üzerinden güncel hava durumu bilgilerini çeker.

- Detaylı Bilgi: Sıcaklık, nem oranı, rüzgar hızı ve gökyüzü durumunu gösterir.

- Akıllı Renklendirme: Sıcaklığa göre dinamik renk değişimi (Sıcaksa kırmızı, soğuksa mavi).

- Kullanıcı Dostu: Şehir ve ilçe bazlı kolay sorgulama ve döngüsel kullanım imkanı.

## 🛠️ Kurulum ve Çalıştırma
1. Depoyu klonlayın:
- git clone https://github.com/vaxen404/weather-app.git
- cd weather-app

2. Bağımlılıkları yükleyin:
- npm install

3. Uygulamayı başlatın:
- npx tsx src/app.ts

## 📦 Teknolojiler
- TypeScript: Güvenli ve ölçeklenebilir kod yapısı.

- Dotenv: API anahtarı güvenliği ve çevre değişkeni yönetimi.

- Chalk: Terminal çıktılarını profesyonelce renklendirme.

- Readline (Node.js): Etkileşimli terminal girişleri.
